# @lukekaalim/act-doc

```sh
npm i @lukekaalim/act-doc
```

_really_ barebones documentation component library for [@lukekaalim/act](https://act.luke.kaal.im).

Write some markdown, annotate with components, and bundle with
your tool of choice!
